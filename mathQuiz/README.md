# Math Speed Quiz 🚀

A futuristic, high-paced single-player math quiz game built with Vanilla JavaScript, HTML, and CSS.

![Math Speed Quiz](https://via.placeholder.com/800x450.png?text=Math+Speed+Quiz+Cyber+Arcade) <!-- Replace with your own screenshot -->

## ✨ Features
- **60-Second Blitz**: Race against the clock to solve as many problems as possible.
- **Smart Math Engine**: Randomized addition, subtraction, multiplication, and clean-result division.
- **Futuristic UI**: A "Modern Chalkboard" aesthetic with neon blue accents and glow effects.
- **Immersive Effects**:
  - 3D Spinning Cube with math symbols.
  - Floating background elements for a digital matrix feel.
  - Urgent timer pulse when less than 10 seconds remain.
- **Instant Feedback**: Visual cues for correct (Green) and wrong (Red) answers.

## 🕹️ How to Play
1. Click **START MISSION**.
2. Solve the math problem displayed in the center.
3. Select the correct answer from the three options provided.
4. Each correct answer gives you **+2 points**.
5. Aim for the highest score before the 60-second timer runs out!

## 🛠️ Technology Stack
- **HTML5**: Semantic structure.
- **CSS3**: Advanced animations, 3D transforms, and custom styling.
- **Vanilla JavaScript**: Game logic, state management, and DOM manipulation.
